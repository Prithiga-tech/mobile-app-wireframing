# Mobile App Wireframing

## Project Overview
This project contains a low-fidelity wireframes for a Student Study Planner mobile application.

## Objective
To understand the early-stage mobile app planning, user research, and user flow creation.

## User Research
Students often forget assignments and deadlines. Many students need reminders and simple task management tools for real life scenarios.

## User Persona
Name: Prithiga  
Age: xx  
Occupation: College Student  

Goals:
- Track assignments
- reminders for task
- Manage study schedule
- Improve productivity

Problems:
- Forgetting deadlines
- Poor time management

## User Flow
Open the App → Login to App → Home Screen will appear → Add Task to it → Save the Task → View the Tasks

## Wireframes Included Screens
- Login Screen
- Home Screen
- Add Task Screen
- Task List Screen

## Tools Used
- Figma
- GitHub

## Outcome
This project helped in understanding the basics of:
- User research
- User flow creation
- Low-fidelity wireframing
- Mobile app planning using Figma
